🎮 SlayTheSpire2_MultiplayerMods
杀戮尖塔 2 自定义联机卡牌 Mod 

📌 项目简介
本项目是为《杀戮尖塔 2》制作的自定义联机卡牌 Mod，目前正在开发崩坏：星穹铁道主题的专属内容，后续会持续更新更多联机玩法与角色卡牌。

🚀 安装教程
从 main 分支下载 Honkai_Star_Train 文件夹
找到《杀戮尖塔 2》的游戏根目录
在根目录中新建名为 mods 的文件夹
将下载的 Honkai_Star_Train 文件夹复制到 mods 目录下
启动游戏即可自动加载 Mod

🎮 联机说明
与好友联机游玩时，所有参与的玩家都必须安装该 Mod
请确保所有玩家的 Mod 版本完全一致，避免联机异常
